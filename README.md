# ▶ Frontend

```powershell
cd frontend
npm install
npm run dev
```

Runs on

```
http://localhost:5173
```

---

# ▶ Backend

```powershell
cd backend
.\.venv\Scripts\Activate.ps1

uvicorn app.main:app --reload
```

Runs on

```
http://127.0.0.1:8000
```

Swagger API

```
http://127.0.0.1:8000/docs
```

Health Check

```
http://127.0.0.1:8000/health/
```
